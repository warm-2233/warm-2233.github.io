flex 布局-弹性布局

Flex 布局教程：语法篇 - 阮一峰的网络日志 (ruanyifeng.com)

就是将display设置为flex的任何元素。

display: flex;

在一个元素设置了flex后，其内子元素的float clear vertical-align 属性将失效。

在display为flex的元素成为容器，其内子元素称为项目 。

flex原理：通过给父级元素添加flex属性，来控制子元素位置排列的方式。

容器操作

  • flex-direction 设置主轴方向。
  • justify-content 设置主轴上的子元素排列方式。
  • flex-wrap 设置子元素是否换行。
  • align-content 设置侧轴上的子元素的排列方式（多行）。
  • align-items 设置侧轴上的子元素的排列方式（单行）。
  • flex-flow 复合属性。相当于同时设置了 flex-direction flex-wrap 。

主轴/侧轴（X/Y）

默认情况下，以左上角开始，水平向右为主轴，垂直向下的为侧轴。

在flex 布局中子元素 时跟着主轴排列的，主轴方向决定了项目的布局。

使用 flex-direction 设置主轴，那么剩下的便是侧轴了。

属性值 说明
row 默认值，从左到右
row-reverse 从右到左
column 从上到下
column-reverse 从下到上 
<style>
    .flexBox {
        background-color: aqua;
        width: 80%;
        height: 300px;
        display: flex;
        flex-direction: row; /*默认值*/
    }

    .item {
        background-color: darkgrey;
        width: 300px;
        height: 200px;
    }
</style>

<body>
    <div class="flexBox">
        <span class="item"> 1 </span>
        <span class="item"> 2 </span>
        <span class="item"> 3 </span>
    </div>
</body>

主轴上项目排列

justify-content 设置主轴上的子元素排列方式。在使用前要先确定主轴方向。

属性值 说明
flex-start 默认值，从头部开始排列，在主轴为默认值时从左到右排列。
flex-end 从尾部开始排列，
center 沿主轴居中对齐
space-evenly 将每个元素的间隔相等，
space-around 平均分配剩余空间
space-between 两端对齐，项目之间的间隔相等，两侧贴边

flex 布局不换行

默认情况下，所有项目都在一行上，即默认不换行，当子元素超过容器，将会缩小子元素的宽度来使其始终在一行。此时width属性将失效。使用 flex-wrap 定义是否换行。

属性值 说明
no-wrap 默认值。规定弹性项目不会换行。
wrap 可以换行
wrap-reverse 以反方向换行

子元素垂直居中

align-items 设置侧轴上的子元素的排列方式。若你设置了flex-direction 为 column ，那么侧轴就是 row(X轴)。

align-items 只能适用于单行项目上，即子元素不能换行。

属性值 说明
stretch 默认值。元素被拉伸以适应容器。设置height 将以其高度值为准
flex-start 从侧轴开头开始排列
flex-end 从侧轴结尾开始排列
center 延侧轴居中排列
baseline 项目的第一行文字的基线对齐

侧轴多行操作

align-content 设置侧轴上的子元素的排列方式**（只适用于多行）**。

属性值 说明
stretch 默认值。元素被拉伸以适应容器。
flex-start 从侧轴开头开始排列
flex-end 从侧轴结尾开始排列
center 延侧轴居中排列
space-between 两端对齐，空间平均分布
space-around 平均分配剩余空间，中间的留空时两边的两倍
space-evenly 将每个元素的间隔相等，

对于子元素单行多行，可以使用 flex-wrap 控制，不管你有N个项目子元素只要将flex-wrap 设置为no-wrap 那它就是单行。

总是喜欢偷懒

flex-flow 复合属性。相当于同时设置了 flex-direction flex-wrap 。

#dia1 {
    flex-direction: row; 
    flex-wrap: wrap;
}
/*等价于*/
#div1 {
    flex-flow: row wrap;
}

需要注意flex-flow可以省略参数，甚至可以不按顺序；省略参数就代表有默认值，请千万小心这一点，它将覆盖你以前定义的属性。只是非常常见的失误。

项目操作

  • order 定义项目的排序顺序。
  • flex-grow 定义项目的放大比例。
  • flex-shrink属性定义了项目的缩小比例。
  • flex-basis 定义项目大小。
  • flex 属性简写。
  • align-self 定义单个项目的排序方式。

项目排序

order 定义项目的排序顺序。

.item {
    order: 0;
}

默认值为0 ，数值越大排序越靠前。

项目放大

flex-grow 定义项目的放大比例，默认值为0，表示还有剩余空间也部放大。

如果所有项目的flex-grow属性都为1，则它们将等分剩余空间（如果有的话）。如果一个项目的flex-grow属性为2，其他项目都为1，则前者占据的剩余空间将比其他项多一倍。

项目缩小

flex-shrink属性定义了项目的缩小比例，其默认值欸1；当其值设置为0时，那么该项目将不会缩小。数值越大缩小比例相对越大。

.item:nth-child(1) {
    flex-shrink: 0;
}

确定项目大小

flex-basis 定义项目大小。默认值为auto，即项目原本的大小，设置这个属性后width属性将失效（flex的很多属性的优先级都比width高，都会覆盖width）。

可以将flex-basis当作width来使用，定义项目的宽度，结合flex-shrink: 0;属性实现固定大小。

当设置了flex-basis属性的新项目放大时，将时flex-basis加上 flex-grow 放大的值。

属性简写

flex 属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。

.item {
    flex: 1;
}

特立独行

align-self属性允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。

该属性可能取6个值，除了auto，其他都与align-items属性完全一致。

<style>
    .flexBox {
        background-color: aqua;
        width: 90%;
        height: 300px;
        display: flex;
    }

    .item {
        background-color: darkgrey;
        width: 300px;
        height: 200px;
        margin: 2px;
        border: 1px solid black;
    }

    .item:nth-child(2){
        align-self: center;
    }
</style>

<body>
    <div class="flexBox">
        <span class="item"> 1 </span>
        <span class="item"> 2 </span>
        <span class="item"> 3 </span>
    </div>
</body>
