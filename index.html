fusuma😃

安装

需要 nodejs 环境

npm install fusuma

// 全局安装
npm install fusuma -g

目录结构

仅需要目录。slides

.
├── .fusumarc.yml
├── slides  // 存放演示文档，固定
│   └── 0-title.md
└── style.css

开始

// 创建项目
mkdir youProject

// 进入
cd youProject

// 初始化
npm init
npx fusuma init

// 写文档
cd slides // slides 存放演示文档的目录，在里面有 0-title.md 文件，在里面写markdown即可

// 演示
npx fusuma start    // 开启一个本地服务器 http://localhost:8080

配置文件

.fusumarc.yml如果配置文件不存在，Fusuma 可以运行 

# 以下是默认配置项
meta:
  url:
  title:
  thumbnail:
  description:
  sns:
    - twitter
slide:
  loop: true
  sidebar: true
  targetBlank: true
  showIndex: true
  isVertical: false
  code:
    languages:
      - javascript
    plugins: []
    theme: default
  chart: false
  math: false
extends:
  js:
  css: ./style.css
  webpack:
build:
  ssr: true
  useCache: true
  publicPath: /
